# A-volar
## Objetivos del projecto
En este projecto nuestro objetivo es programar un dron DJI TELLO para que vuele alrededor de unas columnas en nuestro instituto, Para programar este dron vamos a usar la aplicación droneblocks, que es una aplicación de programación para drones que usa bloques para programar de forma simple.
